# Golf Score Keeper

コンピュータサイエンス学習プラットフォーム Recursion のAngularコースの課題として作成したプロジェクト「Golf Score Keeper」です。

## プロジェクトの概要

ゴルフスコアを追跡するアプリケーションです。
コース名と、ホール数を選択することで、スコア表が出現します。
ホール数とプレイヤーの増減を動的に変更できる形となっています。

スコア表に関して、
プレイヤーの打数をパー列とくらべて、赤と青の色分けがされます。
